# SUAppClient マニュアル

SUAppClient（学生向けクライアントアプリケーション）のマニュアルをここに配置します。

## マニュアルの追加方法

1. このディレクトリにMarkdownファイル（`.md`）を追加します
2. 親ディレクトリの`index.md`にリンクを追加します

## ファイル構成例

```
suapp-client/
├── README.md              # このファイル
├── はじめに.md
├── 基本操作.md
├── 機能説明/
│   ├── プレゼンテーション.md
│   ├── フィードバック.md
│   └── ...
└── トラブルシューティング.md
```
