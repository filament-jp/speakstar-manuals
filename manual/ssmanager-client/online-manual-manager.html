1.はじめに

1.1 管理画面とは？

SPEAK STAR管理画面は、SPEAK STARアプリケーションの運用・管理を行うためのインターネット上の管理システムです。
生徒の学習状況を把握し、アカウント管理やテーマ設定などを行うことで、効果的な学習環境を提供することを支援します。

1.2 管理権限について

SPEAK STAR管理画面では、権限に応じて以下の2種類の機能にアクセスできます：

  1. 学校管理者

    学校単位の管理

    • 自校の情報確認
    • 教師・生徒アカウントの管理
    • 生徒の学習データリセット
    • 各クラスの情報確認
    • テーマ設定の確認
    • システムの利用状況・操作ログの確認
  2. 教師

    クラス・生徒の管理

    • 生徒のモニタリング
    • 各クラスの情報確認
    • 生徒の学習状況確認
    • 生徒のパスワード管理
    • テーマ設定の確認

2. ログイン

2.1 初回ログイン

アカウント発行後、初めてログインする際の手順です。

  1. 1
    初期パスワードでログイン

    管理者から発行された初期パスワードでログインします。

  2. 2
    パスワード変更画面への自動遷移

    初回ログイン時は自動的にパスワード変更画面に遷移します。

  3. 3
    新しいパスワードの設定
    • 現在のパスワード（初期パスワード）を入力
    • 新しいパスワードを2回入力
    • パスワード要件を満たしていることを確認
  4. 4
    設定完了

    パスワード変更が完了すると、ダッシュボード画面に自動遷移します。

パスワード要件
  1. 8文字以上
  2. 大文字・小文字を含む
  3. 数字を含む
  4. 特殊文字を1つ以上含む
パスワード有効期限
  • セキュリティの都合上、作成したパスワードは90日間のみ有効です。
    (期限切れ時は自動的にパスワード変更画面に遷移します)

2.2 通常ログイン

2回目以降のログイン手順です。

  1. 1
    管理画面にアクセス

    ブラウザで管理画面のURLにアクセスします。

  2. 2
    ログイン情報の入力

    メールアドレスとパスワードを入力します。

  3. 3
    ログイン完了

    認証が成功すると、ダッシュボード画面に自動遷移します。

注意！
  • メールアドレスの大文字・小文字は区別されます
  • スペースが入力されていると認証が失敗します
  • パスワードは他の人に教えないよう注意してください
  • 共有端末の場合、使用後は必ずログアウトしてください
  • 連続でパスワード認証に失敗すると、アカウントが一時的にロックされます（※回数に応じて最大約15分間）

2.3 パスワードを忘れた場合

パスワードを忘れた場合は、以下の手順でリセットできます。

  1. 1
    「パスワードをお忘れですか？」をクリック

    ログイン画面の「パスワードをお忘れですか？」リンクをクリックします。

  2. 2
    メールアドレスの入力

    登録済みのメールアドレスを入力し、送信ボタンをクリックします。

  3. 3
    初期パスワードの受信

    入力したメールアドレス宛に初期パスワードが送信されます（有効期限：24時間）。

  4. 4
    初期パスワードでログイン

    メールに記載された初期パスワードでログインします。自動的にパスワード変更画面に遷移します。

  5. 5
    新しいパスワードの設定

    パスワード要件を満たす新しいパスワードを設定して完了です。

3. 学校管理者機能

3.1 ダッシュボード

学校管理者ダッシュボードでは、自校の状況を確認できます。

主要機能

SPEAK STARの学習は以下の4ステップで構成されています。

  1. 学校情報表示

    • 学校名、学校コード、所在地、連絡先、説明を表示
    • 変更不可の読み取り専用表示

  2. 学校全体サマリー

    • シート使用状況、AIクレジット使用量、サービス期間を表示
    • 使用量と上限の関係を視覚的に確認

    AIクレジット表示

    AIクレジットは「使用量 / 上限」の形式で表示されます。

    • 表示例: 30000 / 50000
    • これは30,000クレジットを使用し、上限が50,000クレジットであることを示します

  3. 統計情報

    • クラス数、教師数、生徒数（有効）などの数値情報
    • 表示される生徒数は有効状態の生徒のみをカウント

  4. グラフ表示エリア

    • 統計情報を視覚的に表示
    • データの傾向を分かりやすく確認
注意！
  • AIクレジット使用量: 上限に近づいている場合はシステム管理者に連絡してください
FAQ
Q AIクレジット使用量が上限に近づいています
A
システム管理者に連絡して、追加クレジットの配布を依頼してください。
Q 統計情報が更新されません
A
ページを再読み込みしてください。それでも更新されない場合は、システム管理者に連絡してください。

3.2 クラス管理サマリー

各学年・クラスごとの生徒数とAIクレジット使用量などの概要情報を確認できます。

主要機能

  1. クラス一覧表示

    • 学年別、クラス別の一覧表示
    • 各クラスの生徒数、AIクレジット使用量を表示

  2. 統計情報

    • 学年ごとの生徒数合計
    • クラスごとのAIクレジット使用状況
    • 使用率の視覚的表示

3.3 教師・生徒アカウント管理

自校の教師・生徒アカウントの管理を行います。

主要機能

  1. アカウント一覧表示

    • 教師・生徒のアカウント一覧を表示
    • アカウント種別、名前、メール、クラス、学年、状態を表示

  2. アカウント操作

    • 新規登録、編集、削除機能
    • パスワードリセット機能
    • アカウント状態の変更（有効/無効）

  3. 一括登録機能

    • 一括登録用ポップアップ画面
    • 指定フォーマットのファイルで複数アカウントを一括登録
    • 既存の生徒IDとの重複チェックを実施

  4. 絞り込み・検索機能

    • アカウント種別、クラス、学年、状態での絞り込み
    • 名前、メール、生徒IDでの検索
注意！
  • 生徒ID: 同一学校内で重複しないよう注意してください
  • アカウント削除: 削除されたアカウントは復元できません
  • 状態変更: 無効にしたアカウントはログインできなくなります
FAQ
Q 生徒IDが重複エラーになります
A
同一学校内で同じ生徒IDが使用されています。異なる生徒IDを入力してください。
Q アカウントの状態を変更したい場合は？
A
アカウント一覧の状態列で「有効」「無効」を切り替えることができます。

3.4 テーマ設定確認

学習テーマの確認を行います

主要機能

  1. テーマ一覧表示

    • 学年別のテーマ一覧を表示
    • テーマ名、説明、学年を確認
    • 1ページあたり50件のテーマを表示
    • ページネーション機能あり

  2. テーマ詳細確認

    • 各テーマの詳細情報を確認
    • テーマの内容と学習目標を把握

  3. 閲覧専用

    • テーマの追加・編集・削除は不可
    • システム管理者による設定のみ
FAQ
Q テーマを追加・変更したい場合は？
A
テーマの追加・変更はシステム管理者に依頼してください。学校管理者では変更できません。
Q 学年ごとに異なるテーマが表示されません
A
システム管理者に連絡して、学年別のテーマ設定を確認してください。

4. 教師機能

教師は、担当クラスの生徒管理と学習状況の確認を行います。

4.1 生徒モニタリング

授業中に生徒の状況をリアルタイムでモニタリングできます。

主要機能

  1. リアルタイム状態表示

    • 生徒の状態表示（アクティブ、アイドル、オフライン）
    • リアルタイムでの対話内容確認
    • 並び順: 学年（昇順）→ クラス（昇順）→ 出席番号（昇順）

  2. 表示切替

    • カード表示/リスト表示の切り替え
    • 表示サイズの変更（小・中・大）

4.2 クラス一覧

担当クラスの概要情報を確認できます。

主要機能

  1. クラス一覧表示

    • 担当クラスの一覧を表示
    • クラス名、学年、生徒数を確認

4.3 生徒管理

担当クラスの生徒情報を管理し、学習状況を確認できます。

主要機能

  1. 生徒一覧表示

    • 担当クラスの生徒一覧を表示
    • 生徒名、出席番号、学習状況を確認
    • 並び順: 学年（昇順）→ クラス（昇順）→ 出席番号（昇順）
    • 生徒名をクリックすると、生徒詳細画面が別タブで開きます
    • 複数の生徒を同時に確認可能で、一覧画面に簡単に戻れます

  2. 学習状況確認

    • 各生徒の学習進捗を確認
    • チャット回数、プレゼンテーション回数を表示

  3. パスワード管理

    • 生徒のパスワードリセット機能
    • パスワード状態の確認（初期パスワード/変更済み）

  4. 生徒詳細確認

    • 個別の生徒の学習状況を詳細に確認
    • 対話情報、プレゼン情報の表示
    • 音声再生、フィードバックコメント等の詳細情報
    • 生徒詳細画面は別タブで開くため、一覧画面に簡単に戻れます

    対話内容の表示:

    • 最初のお題メッセージ（会話のきっかけ）は表示されません
    • 生徒の実際の発言のみが表示・カウントされます

4.4 テーマ設定確認

担当クラスに割り当てられているテーマを確認できます。

主要機能

  1. テーマ一覧表示

    • 担当クラスに割り当てられているテーマコレクションとテーマの一覧確認
    • 学年ごとのテーマ割り当て状況
    • 1ページあたり50件のテーマを表示（ページネーション操作を減らし、一覧性を向上）

  2. 閲覧専用

    • テーマの追加・編集・削除は不可
    • システム管理者による設定のみ
注意！
  • パスワードリセット: リセット後は新しい初期パスワードでログインし、必ずパスワードを変更してください
  • 印刷機能: 初期パスワードアカウントのみ選択可能なオプションがあります

5. トラブルシューティング

5.1 ログイン・認証に関する問題

Q ログインができません
A

以下の点を確認してください：

  1. ログイン情報の確認
    • メールアドレスとパスワードが正しく入力されているか
    • 大文字・小文字も区別されます
  2. パスワードの確認
    • パスワードを忘れた場合は「パスワードをお忘れですか？」からリセット
    • リセットリンクがメールに送信されます
  3. アカウントの状態確認
    • アカウントが有効になっているか
    • 管理者に確認を依頼

5.2 アカウント管理に関する問題

Q 一括登録でエラーが発生します
A

以下の点を確認してください：

  1. ファイル形式
    • 正しい形式で保存されているか
    • 文字化けが発生していないか
  2. 必須項目
    • アカウント種別、名前、学年（生徒の場合）、クラス（生徒の場合）
    • 出席番号（生徒の場合）が入力されているか
  3. データ形式
    • メールアドレス：正しい形式か
    • 生徒ID：半角英数字で、既存IDと重複していないか
    • 学年：数値（1〜6）で入力されているか

5.3 データ表示・更新に関する問題

Q プレゼンしたいお題が表示されていない
A

以下の点を確認してください：

  1. ブラウザの確認
    • 推奨・対応ブラウザを使用しているか
    • キャッシュをクリアして再読み込みを行ったか
  2. 権限の確認
    • 適切な権限（教師・管理者など）でログインしているか
    • 必要に応じて管理者に権限設定の確認を依頼する
  3. システムエラーの可能性
    • 一時的なシステム障害の可能性がある
    • 時間をおいて再度アクセスしても解消しない場合は、管理者へ問い合わせる

5.4 システムエラーに関する問題

Q エラーメッセージが表示されます
A

エラーの種類に応じて、次の点を確認してください：

  1. 認証エラー
    • メールアドレス・パスワードが正しく入力されているか
    • セッションの有効期限が切れていないか（再ログインを試す）
  2. 権限エラー
    • アクセス権限がない機能にアクセスしていないか
    • 必要な権限が付与されているか、管理者に確認を依頼する
  3. システムエラー
    • 一時的なシステム障害の可能性がある
    • 時間をおいて再試行し、それでも解決しない場合はサポート窓口に連絡する
  4. ネットワークエラー
    • インターネット接続が安定しているか
    • 学校内ネットワークやプロキシ設定などに問題がないか確認する
